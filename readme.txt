== Description ==

Plugin DS AdRotator allows you to store banners and display them in a given zone.

== Installation ==

1. Upload `ds-adrotator.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php if(function_exists('ds_show_banners')) {ds_show_banners(zoneID);} ?>` in your templates
